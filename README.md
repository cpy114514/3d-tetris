# 3D Tetris Unity

A Unity 2022 3D Tetris prototype using a 2D gameplay view inside a 3D cuboid world.

## Project

- Unity version: 2022.3.62f3
- Main scene: `Assets/Scenes/GamePlay.unity`
- Main gameplay script: `Assets/Scripts/ThreeDTetris/ThreeDTetrisGame.cs`

## Controls

- `A / D`: move left and right
- `W`: rotate piece
- `S / Space`: release piece and soft drop
- `Q / E`: rotate the cuboid
- `P`: pause

## Editing

Open `GamePlay.unity`, select `Editable Cuboid Tetris`, and use the Inspector to edit board settings, materials, UI references, preview colors, and scene setup tools.
