# 3D Tetris Unity
A Unity 2022 3D Tetris prototype using a 2D gameplay view inside a 3D cuboid world.
## Controls

- A / D: move left and right
- W: rotate piece
- S / Space: release piece and soft drop
- Q / E: rotate the cuboid
- Esc: pause panel
